# sql-check-constraint-compiler

SQL Check Constraint Compiler

## Setup

1. Create Virtual Environment

```bash
python3 -m venv venv
```

2. Activating Virtual Environment

On Windows:

```bash

.\venv\Scripts\activate
```

On macOS and Linux:

```bash
source venv/bin/activate
```

3. Installing Dependencies

```bash
pip install -r requirements.txt
```
